## Тема
Виды процедур SPA салона — процедуры

## Описание
Настольное приложение, разработанное с использованием .NET MAUI и Entity Framework Core.
Приложение реализует хранение и отображение видов процедур SPA салона и процедур,
относящихся к каждому виду.

## Архитектура
- Clean Architecture
- MVVM
- CQRS (MediatR)
- UnitOfWork
- Dependency Injection

## Технологии
- .NET 8
- .NET MAUI
- Entity Framework Core
- SQLite
- MediatR

## Функционал
- Просмотр списка видов процедур
- Просмотр процедур выбранного вида
- Подсветка процедур с малой длительностью
- Хранение данных в базе данных SQLite

## Структура проекта
- **SpaSalon.Domain** — сущности предметной области
- **SpaSalon.Application** — бизнес-логика и интерфейсы
- **SpaSalon.Infrastructure** — работа с БД (EF Core)
- **SpaSalon.Maui** — пользовательский интерфейс
